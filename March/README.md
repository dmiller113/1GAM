1GAM entry for March: Pong!

Written in [elm-lang](http://www.elm-lang.org)

#### Installation
1. ```yarn```
2. ```yarn run elm package install```
3. ```yarn run elm reactor```
4. Navigate to `127.0.0.1:8000`

#### Controls
- W, S for Player 1
- Arrow keys for Player 2
- Space, R to reset
